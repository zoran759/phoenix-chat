<http://work.stevegrossi.com/2016/07/11/building-a-chat-app-with-elixir-and-phoenix-presence/>

## Installation

Requires Erlang and Elixir.

```
$ git clone git@github.com:stevegrossi/phoenix_chat.git
$ cd phoenix_chat
$ mix deps.get
$ mix ecto.create
$ mix phoenix.server
```
